    带有else的条件语句：


     变量名 = 非比较值
     if 变量名 == 比较值:
       print("The condition is True.")
     else:
       print("The condition is False.")


    示例输出结果：
       The condition is False.


    内涵：
    通过if判断变量指向的值与比较值之间的关系。若该关系与比较符判断一致，则输出1（True），否则输出0（False）。当输出1时，会执行':'后缩进的（归属该条件语句的）语句；当输出0时，会执行'else:'后缩进的语句。
    //python通过缩进判断代码块的归属关系

    多条件语句：


     变量名 = 比较值02
     if 变量名 == 比较值01:
       print("The first condition is True.")
     elif:
       print("The second condition is True.")
     else:
       print("All the conditions are False.")


    示例输出结果：
       The second condition is False.


    内涵：
    将变量与if和elif语句中的比较值一一对比，向与实际关系相同的条件语句输出1，并执行该条件语句下属的语句。若实际关系不与任何条件语句中的比较符关系重合，则输出0，并执行else语句下属的语句。
